# Backend — Scenario Builder API

This is the .NET Web API starter skeleton for the Backend-Oriented assignment.

## Getting started

```bash
dotnet restore
dotnet run --project Backend.Api
```

The API runs on `http://localhost:5000` by default. Swagger UI is available at `http://localhost:5000/swagger` in development.

## What to implement

Build the full backend from here. See the assignment document for requirements.

`Program.cs` is pre-configured with controllers, Swagger, and TODO comments marking where to add CORS and DI registrations. Everything else — domain models, DTOs, repositories, controllers, architecture — is your design decision.
