# Frontend — React + TypeScript (Vite)

This is the React starter skeleton for the Frontend-Oriented assignment.

## Getting started

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5173` by default.

## Environment variables

| Variable | Default | Description |
|---|---|---|
| `VITE_API_URL` | `http://localhost:5000` | Base URL of the backend API |

The `.env` file is pre-configured for local development. Update it if your backend runs on a different port.

## What to implement

Build the full frontend application from here. See the assignment document for requirements.

The entry point is `src/App.tsx` — React Router is already installed and `<BrowserRouter>` is wired up. Add your routes and components from there.
